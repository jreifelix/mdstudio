Ligado ao Google Calendar!

As reservas são lidas e escritas diretamente no Google Calendar — todos os dispositivos ficam sincronizados automaticamente.
⚠️ A app deve ser acedida via http:// (servidor local ou web hosting). A integração não funciona com file://.
ℹ️ Como configurar — passo a passo ▸
1. Aceda a console.cloud.google.com e crie (ou selecione) um projeto.
2. Vá a APIs e Serviços → Biblioteca e ative a Google Calendar API.
3. Vá a APIs e Serviços → Credenciais → Criar credenciais → ID do cliente OAuth 2.0 → tipo Aplicação Web.
4. Em Origens JavaScript autorizadas adicione o URL onde a app está a ser servida (ex: http://localhost:3000 ou o seu domínio).
5. Copie o ID do cliente gerado e cole no campo abaixo.
